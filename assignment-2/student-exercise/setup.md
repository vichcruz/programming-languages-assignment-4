# Go Setup Instructions

## Mac

1. Install Homebrew if you donâ€™t have it:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. Install Go:

```bash
brew install go
```

3. Verify installation:

```bash
go version
```

## Windows

1. Download the Go installer from https://go.dev/dl/
2. Follow the installer instructions.
3. Verify installation: open CMD or PowerShell and run:

```bash
go version
```

## Optional IDE

- Visual Studio Code with the Go extension is recommended for coding and debugging.
- Alternatively, any text editor can be used with terminal commands.
